# Agent Instructions

## Agent skills

### Issue tracker

Issues are tracked in Jira project `RAP` on `bluereplyexp.atlassian.net`; local issue markdown files are planning drafts until published to Jira. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the default triage role vocabulary unless Jira project `RAP` is configured with different fields or labels. See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses a single-context domain layout rooted at `CONTEXT.md`. See `docs/agents/domain.md`.
